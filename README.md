# FileMergeRenamer

여러 폴더에 흩어진 이미지 파일을 원하는 순서로 병합하고, 일련번호로 일괄 이름 변경하는 GUI 프로그램입니다.

---

## 지원 포맷

`JPG` `JPEG` `PNG` `GIF` `BMP` `WebP` `TIFF` `HEIC` `AVIF`

---

## 설치 및 실행

### EXE 실행 (설치 불필요)
releases 페이지에서 `FileMergeRenamer.exe` 다운로드 후 바로 실행

### 소스코드로 실행
```bash
pip install -r requirements.txt
python main.py
```

---

## 사용 방법

### 1단계 — 폴더 추가

폴더를 추가하는 방법은 두 가지입니다.

- **드래그 앤 드롭** : 탐색기에서 폴더를 파란 점선 영역에 끌어다 놓기 (여러 개 동시 가능)
- **버튼으로 추가** : `＋ 폴더 추가` 버튼 클릭 후 폴더 선택

> 하위 폴더에 이미지가 있으면 자동으로 하위 폴더까지 수집됩니다.

---

### 2단계 — 병합 순서 조정

왼쪽 **폴더 목록**에서 항목을 드래그하여 순서를 원하는 대로 변경할 수 있습니다.

오른쪽 **병합 미리보기**에서 최종 파일 순서와 변경될 이름을 확인할 수 있습니다.

폴더를 삭제하려면 항목 선택 후 `선택 삭제` 또는 전체를 지우려면 `전체 삭제`를 클릭합니다.

---

### 3단계 — 파일명 옵션 설정

| 옵션 | 설명 |
|------|------|
| **숫자만** | 001, 002, 003 ... 형태로 이름 변경 |
| **접두사 + 숫자** | image_001, image_002 ... 형태로 이름 변경 |
| **자릿수** | 숫자 앞 0 패딩 자릿수 선택 (1 ~ 00001) |
| **시작 번호** | 번호를 시작할 숫자 (기본값: 1) |
| **폴더 내 정렬** | 각 폴더 안에서 파일을 정렬하는 기준 선택 |

**정렬 기준**
- `파일명순` — 파일명 기준 자연 정렬 (기본값)
- `수정일순` — 마지막 수정 날짜 기준
- `생성일순` — 파일 생성 날짜 기준
- `크기순` — 파일 크기 기준

---

### 4단계 — 출력 설정

| 옵션 | 설명 |
|------|------|
| **저장 위치** | 결과 파일을 저장할 경로 선택 |
| **폴더로 저장** | 지정한 폴더에 이름이 변경된 파일로 저장 |
| **ZIP으로 저장** | 결과를 ZIP 파일 하나로 압축 저장 |
| **원본 보존 (복사)** | 체크 시 원본 파일 유지, 해제 시 원본 이동 (삭제) |

> ZIP 저장 시 저장 경로에 `.zip` 확장자를 직접 입력하거나, 폴더를 선택하면 폴더명으로 ZIP이 생성됩니다.

---

### 5단계 — 실행

`실 행` 버튼을 클릭하면 작업이 시작됩니다.

- 진행률 표시줄에서 처리 현황을 확인할 수 있습니다.
- 작업 중 `취 소` 버튼을 클릭하면 즉시 중단됩니다. (생성 중이던 ZIP 파일은 자동 삭제)
- 완료 시 저장 위치가 표시된 완료 메시지가 나타납니다.

## 사용 예시

**상황** : `vol.1`, `vol.2`, `vol.3` 폴더에 각각 이미지가 있고, 하나로 합쳐서 `001.jpg`, `002.jpg` ... 로 이름을 바꾸고 싶은 경우

1. 세 폴더를 드래그 앤 드롭으로 추가
2. 폴더 목록에서 순서 확인 (드래그로 조정)
3. 자릿수 `001` 선택, 시작 번호 `1`
4. 저장 위치 선택, `폴더로 저장` 선택
5. `원본 보존` 체크 후 `실 행`

결과: 지정 폴더에 `001.jpg`, `002.jpg`, `003.jpg` ... 생성
